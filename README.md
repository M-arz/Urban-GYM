# UrbanGym

Sistema basado en arquitectura de microservicios para la gestión de gimnasios.

## Tecnologías

- Node.js
- NestJS
- PostgreSQL
- Docker
- GitHub Actions

## Microservicios

- Member Service
- Booking Service
- Billing Service
- Workout Service
- Recommendation Service

## Arquitectura

El sistema utiliza una arquitectura de microservicios conectados mediante un API Gateway.
